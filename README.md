# English Learning Journey for Frontend Developers

## Giới thiệu / Introduction

Đây là hành trình học tiếng Anh của tôi, một frontend developer. Repository này ghi lại toàn bộ quá trình học tập từ beginner đến thành thạo.

This is my English learning journey as a frontend developer. This repository documents my complete learning process from beginner to proficiency.

## Mục tiêu / Goals

- **Reading**: Đọc hiểu tài liệu kỹ thuật, documentation, technical articles
- **Writing**: Viết commit messages, PR descriptions, technical documentation
- **Listening**: Nghe hiểu tech talks, conferences, team meetings
- **Speaking**: Giao tiếp với team quốc tế, thuyết trình, phỏng vấn

## Phương pháp học / Learning Method

1. **Daily Practice**: Học mỗi ngày 30-60 phút
2. **Context-based**: Học qua ngữ cảnh lập trình thực tế
3. **4 Skills Balance**: Cân bằng cả 4 kỹ năng
4. **Practical Focus**: Tập trung vào thực hành, không chỉ lý thuyết
