# Hướng dẫn học tập / How to Study

## Lộ trình học / Learning Path

### Phase 1: Foundation (Tháng 1-2) - Beginner
- **Mục tiêu**: Nắm vững nền tảng cơ bản
- **Nội dung**:
  - Alphabet & Pronunciation
  - Basic vocabulary (100-200 từ)
  - Simple sentences
  - Present tense
  - Tech terms cơ bản

### Phase 2: Building (Tháng 3-4) - Elementary
- **Mục tiêu**: Đọc hiểu tài liệu đơn giản
- **Nội dung**:
  - Reading documentation
  - Writing commit messages
  - Past & Future tenses
  - Tech vocabulary (500+ từ)

### Phase 3: Practice (Tháng 5-6) - Pre-Intermediate
- **Mục tiêu**: Giao tiếp cơ bản
- **Nội dung**:
  - Technical discussions
  - Code reviews
  - Listening to tech talks
  - Speaking practice

---

## Lịch học hàng ngày / Daily Schedule

### 30-45 phút mỗi ngày (30-45 minutes daily)

#### Sáng (Morning) - 15 phút
- ☀️ Ôn lại từ vựng ngày hôm trước (5 phút)
- 📖 Học 5 từ mới (10 phút)

#### Trưa/Chiều (Afternoon) - 15 phút
- 📝 Làm bài tập trong lesson (15 phút)

#### Tối (Evening) - 15 phút
- 🎧 Nghe và lặp lại (10 phút)
- ✍️ Viết câu với từ mới học (5 phút)

---

## Phương pháp học / Study Methods

### 1. **Spaced Repetition** (Lặp lại ngắt quãng)
- Học từ mới: Ngày 1
- Ôn lại: Ngày 2, 4, 7, 14, 30
- Dùng app: Anki, Quizlet

### 2. **Contextual Learning** (Học qua ngữ cảnh)
- Đọc code có comment tiếng Anh
- Xem documentation gốc
- Đọc tech blog/article

### 3. **Active Practice** (Thực hành tích cực)
- Nói to các từ vựng
- Viết commit message bằng tiếng Anh
- Comment code bằng tiếng Anh

### 4. **Immersion** (Đắm mình)
- Đổi VSCode sang tiếng Anh
- Xem YouTube tech video có subtitle
- Tham gia Discord/Slack tiếng Anh

## Công cụ hỗ trợ / Tools

### Phát âm (Pronunciation)
- 🔊 **Google Translate**: Text-to-speech
- 🗣️ **Forvo.com**: Native speakers
- 📱 **ELSA Speak**: AI pronunciation coach

### Từ vựng (Vocabulary)
- 📚 **Anki**: Flashcard spaced repetition
- 📝 **Quizlet**: Study sets
- 📖 **Notion**: Personal vocabulary database

### Nghe (Listening)
- 🎥 **YouTube**: Tech talks with subtitles
- 🎧 **Podcast**: Syntax.fm, JavaScript Jabber
- 📺 **Conference**: ReactConf, VueConf videos

### Đọc (Reading)
- 📄 **MDN Web Docs**: Clear technical English
- 📰 **Dev.to**: Developer articles
- 📚 **GitHub README**: Well-written projects

### Viết (Writing)
- ✍️ **Grammarly**: Grammar checker
- 📝 **Hemingway Editor**: Clarity checker
- 💬 **ChatGPT**: Practice conversations

### Nói (Speaking)
- 🗨️ **HelloTalk**: Language exchange
- 👥 **Discord servers**: Tech communities
- 🎤 **Shadowing**: Repeat after videos

## Tips quan trọng / Important Tips

1. ✅ **Consistent > Intensity**: Học đều 30 phút/ngày tốt hơn 3 giờ/tuần
2. ✅ **Mistakes are OK**: Sai là cách học, đừng sợ mắc lỗi
3. ✅ **Use it daily**: Dùng tiếng Anh trong công việc hàng ngày
4. ✅ **Track progress**: Ghi lại tiến bộ để có động lực
5. ✅ **Have fun**: Học qua nội dung bạn thích (game, tech talk, etc.)

## Đánh giá tiến bộ / Progress Evaluation

### Mỗi tuần (Weekly)
- Review từ vựng đã học

### Mỗi tháng (Monthly)
- Đánh giá 4 kỹ năng
- Điều chỉnh lộ trình nếu cần
- Celebrate achievements! 🎉
